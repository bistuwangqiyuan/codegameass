# GameCode Lab 用户指南

欢迎来到 GameCode Lab！本指南将帮助你快速上手，充分利用平台的所有功能。

## 🎮 快速入门

### 1. 开始学习之旅

#### 游客模式（推荐新手）

1. 访问 [GameCode Lab](https://your-domain.com)
2. 点击 **"免费试用 30 天"** 按钮
3. 系统自动创建游客账号
4. 立即开始学习！

**游客模式特点：**
- ✅ 无需注册，即刻开始
- ✅ 30 天完整功能访问
- ✅ 所有课程和挑战
- ✅ AI 助教全程陪伴
- ✅ 保存学习进度

#### 注册正式账号

1. 点击导航栏的 **"登录/注册"**
2. 选择 **"注册"** 标签
3. 填写：
   - 用户名（唯一标识）
   - 昵称（显示名称）
   - 邮箱
   - 密码（至少 6 位）
4. 点击 **"注册"** 完成

**正式账号优势：**
- 🔐 永久数据保存
- 🏆 完整成就系统

## 💻 代码编辑器使用

### 界面布局

```
┌─────────────────────────────────────────┐
│ 工具栏: [HTML] [CSS] [JS] | 运行 保存   │
├──────────────────┬──────────────────────┤
│                  │                      │
│   代码编辑区      │      实时预览窗口     │
│                  │                      │
│   支持语法高亮    │   即时显示效果       │
│   自动补全       │                      │
│                  │                      │
├──────────────────┴──────────────────────┤
│ 控制台: 显示日志和错误信息                │
└─────────────────────────────────────────┘
```

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + S` | 保存代码 |
| `Ctrl + Enter` | 运行代码 |
| `Ctrl + /` | 注释/取消注释 |
| `Ctrl + F` | 查找 |
| `Ctrl + H` | 替换 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Shift + Z` | 重做 |
| `Tab` | 缩进 |
| `Shift + Tab` | 取消缩进 |

### 常用功能

#### 1. 运行代码
点击 **"运行"** 按钮或按 `Ctrl + Enter`，代码会在右侧预览窗口实时显示。

#### 2. 保存进度
点击 **"保存"** 按钮或按 `Ctrl + S`，代码会自动保存到云端。

#### 3. 重置代码
点击 **"重置"** 按钮恢复到起始代码（不可撤销）。

#### 4. 查看控制台
点击 **"控制台"** 图标查看 `console.log()` 输出和错误信息。

## 🤖 AI 助教使用指南

### 如何召唤 AI 助教

1. **浮动按钮**: 点击右下角的 ✨ 按钮
2. **快捷键**: 按 `Ctrl + Shift + A`
3. **挑战页面**: 点击顶部 **"AI 助教"** 按钮

### AI 助教能做什么

#### 1. 代码讲解
**使用场景：** 不理解某段代码

**示例对话：**
```
你: 能解释一下这段 CSS 代码吗？
.container {
  display: flex;
  justify-content: center;
}

AI: 当然！这段 CSS 使用了 Flexbox 布局...
```

#### 2. 错误诊断
**使用场景：** 代码运行出错

**示例对话：**
```
你: 我的代码报错了: "Uncaught TypeError: Cannot read property 'value' of null"

AI: 这个错误表示你试图访问一个不存在的元素...
```

#### 3. 代码优化
**使用场景：** 想改进代码质量

**示例对话：**
```
你: 我的代码能优化吗？
[粘贴代码]

AI: 你的代码可以这样改进...
```

#### 4. 学习建议
**使用场景：** 不知道下一步学什么

**示例对话：**
```
你: 我学完了 HTML，接下来应该学什么？

AI: 恭喜完成 HTML！建议你接下来学习 CSS...
```

## ❓ 常见问题

### Q: 游客试用期结束后会怎样？
A: 试用期结束后，需要注册正式账号。你的学习进度会保留，可以继续从断点学习。

### Q: 忘记密码怎么办？
A: 点击登录页面的 "忘记密码"，输入邮箱接收重置链接。

### Q: 可以在手机上使用吗？
A: 可以！我们的网站完全响应式，支持手机、平板访问。但建议在电脑上编写代码以获得最佳体验。

### Q: 代码会保存吗？
A: 是的。点击 "保存" 按钮后，代码会自动保存到云端。即使关闭浏览器，下次登录也能继续。

### Q: AI 助教是免费的吗？
A: 基础对话完全免费。获取挑战提示需要消耗金币（5 金币/次），但金币可以通过学习获得。

### Q: 如何获得更多金币？
A: 完成课程、挑战、每日任务都可以获得金币。发布优质作品获得点赞也能赚取金币。

### Q: 可以删除已发布的作品吗？
A: 可以。在 "我的作品" 页面找到对应作品，点击删除即可。

### Q: 排行榜多久更新一次？
A: 实时更新。每次完成挑战或获得 XP 后，排名会立即刷新。
