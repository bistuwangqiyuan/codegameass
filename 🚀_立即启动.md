## ⚠️ 首次使用需要初始化数据库

如果看到数据库错误，需要先运行数据库迁移：

### 初始化数据库步骤：

1. **访问 Supabase Dashboard**
   - 打开：https://app.supabase.com
   - 登录你的账号

2. **选择项目**
   - 找到项目：`zzyueuweeoakopuuwfau`

3. **运行 SQL 迁移**
   - 点击左侧 **"SQL Editor"**
   - 点击 **"New Query"**
   - 打开本地文件：`supabase/migrations/20250415000000_gamecode_lab_schema.sql`
   - 复制**全部内容**（约 1000 行）
   - 粘贴到 SQL Editor
   - 点击右下角绿色 **"Run"** 按钮
   - 等待 10-15 秒
   - 看到 "Success" 即完成！

4. **刷新页面**
   - 回到 http://localhost:4321
   - 刷新页面（F5）
   - 开始使用！

## 🐛 遇到问题？

### 问题 1：端口被占用
```powershell
# 杀掉占用 4321 端口的进程
netstat -ano | findstr :4321
taskkill /PID <进程ID> /F
npm run dev
```

### 问题 2：数据库连接失败
- 检查 `.env` 文件是否存在
- 确认 Supabase URL 和 Keys 正确
- 运行数据库迁移（见上方步骤）

### 问题 3：AI 不响应
- 系统会自动切换到备用 AI
- 检查网络连接
- 查看浏览器控制台错误信息

### 问题 4：页面空白
```powershell
# 清理缓存重新启动
rm -rf .astro dist
npm run dev
```
